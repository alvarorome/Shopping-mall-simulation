# Simulación centro comercial

Descripción general
Este proyecto desarrolla un modelo de simulación de eventos discretos utilizando Arena Simulation para representar el funcionamiento de un centro comercial compuesto por un restaurante, una tienda de ropa y una zapatería. El objetivo es analizar el flujo de clientes, la utilización de recursos, los tiempos de espera y la eficiencia operativa del sistema bajo diferentes condiciones.
